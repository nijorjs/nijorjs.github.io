What is Nijor ?

Nijor is a component-based framework that allows developers to build websites with ease by breaking down the code into many small components which can be reused throughout the codebase. Thus, Nijor makes developement faster in comparison to plain HTML, CSS and JS.
Nijor compiles down to browser readable code with the help of the nijor compiler .

Feature Nijor Provides :

  • Component Based
  • Frontend Routing
  • Custom Events
  • Easy to implement Dark-Mode

Prerequisites

You need to have some basic knowledge about HTML, CSS and Javascript. You need to have Node.js installed on your computer before using Nijor although you don't need to know anything about NodeJS.
